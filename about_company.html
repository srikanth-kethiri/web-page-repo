This 14-video lab course is the second in a 3-course series on learning how to use Git and GitHub. In this hands-on course on Git, participants will solidify their use of many fundamentals in Git's version control tool. This course demonstrates how the Git ecosystem simplifies setting up source code repositories, and how GitHub makes it easy to use GitHub to collaborate with other developers who use Git. Learn to use the command line to create and manipulate local repositories, while expanding your knowledge of the fundamentals of this version control tool.
